# Templates-Expressjs
 Templates project expressjs
1. install:
    npm install
2. setting .env:   
    Tạo file .env.production nếu sử dụng env riêng
3. usage:
    npm run dev ( use .env.production)
    npm start ( use .env)
# Dependency
    Cors,
    Axios,
    dotenv,
    jsonwebtoken,
    nodemon,
    body-parser,
    socket.io,
    node-localstorage,
    mongodb,
    mongoose
